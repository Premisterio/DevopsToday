# Travel Planner — Backend

FastAPI backend for managing travel projects and artworks sourced from the Art Institute of Chicago API.

## Running locally (without Docker)

**Requirements:** Python 3.11+

```bash
cd be
python -m venv .venv
source .venv/bin/activate      # Windows: .venv\Scripts\activate
pip install -r requirements.txt
uvicorn app.main:app --reload --host 0.0.0.0 --port 8000
```

The database (`travel.db`) is created automatically on first startup.

## Running with Docker Compose (from the repo root)

```bash
docker compose up --build backend
```

## Running with Docker only

```bash
cd be
docker build -t travel-planner-be .
docker run -p 8000:8000 travel-planner-be
```

## Environment variables

Copy `.env` and adjust as needed:

| Variable | Default | Description |
|---|---|---|
| `ARTIC_API_BASE_URL` | `https://api.artic.edu/api/v1` | Art Institute of Chicago API base URL |
| `DATABASE_URL` | `sqlite:///./travel.db` | SQLAlchemy database URL |

## API docs

Swagger UI is auto-generated by FastAPI and available at:

```
http://localhost:8000/docs
```

ReDoc is available at:

```
http://localhost:8000/redoc
```

## Example requests

### Create a project with places

```bash
curl -X POST http://localhost:8000/projects \
  -H "Content-Type: application/json" \
  -d '{
    "name": "My Chicago Art Tour",
    "description": "Visiting the Art Institute",
    "start_date": "2025-06-01",
    "places": [111628, 27992]
  }'
```

### List projects

```bash
curl "http://localhost:8000/projects?page=1&limit=10"
```

### Add a place to a project

```bash
curl -X POST http://localhost:8000/projects/1/places \
  -H "Content-Type: application/json" \
  -d '{"external_id": 111628}'
```

### Mark a place as visited

```bash
curl -X PATCH http://localhost:8000/projects/1/places/1 \
  -H "Content-Type: application/json" \
  -d '{"is_visited": true}'
```

### Update place notes

```bash
curl -X PATCH http://localhost:8000/projects/1/places/1 \
  -H "Content-Type: application/json" \
  -d '{"notes": "Absolutely stunning painting."}'
```

### Delete a project

```bash
curl -X DELETE http://localhost:8000/projects/1
```
